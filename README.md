# End-to-end-Medical-Chatbot-using-Llama2
## Steps to run the project
1. Create the rpository in guthub and git clone the repository using anaconda terminal
2. cd the repository
3. One the terminal type 'code .' # this will open Visio Editor
4. Inside the terminal of the Code Visio type

''' bash
conda create -n mchatbot -y
'''

''' bash
conda activate mchatbot
'''

''' bash
conda install python=3.9 -y
'''

''' bash
pip install -r requirements.txt
'''
